En este video les mostraré como utilizar el Xiaomi Gateway 3 desde Home Assistant, para poder utilizar todos tus sensores Zigbee y Bluetooth LE de la marca Xiaomi o Aqara de forma local y sin depender de la nube.

Comando para activar telnet en el Gateway:

```jsx
{"method":"set_ip_info","params":{"ssid":"\"\"","pswd":"123123 ; passwd -d admin ; echo enable > /sys/class/tty/tty/enable; telnetd"}}
```

Comando para instalar un firmware soportado por la [integración](https://github.com/AlexxIT/XiaomiGateway3) para Home Assistant:

```bash
wget -O /tmp/curl "http://master.dl.sourceforge.net/project/mgl03/bin/curl?viasf=1" && chmod +x /tmp/curl export PATH="$PATH:/tmp"
curl -s -k -L -o /tmp/update.sh https://raw.githubusercontent.com/zvldz/mgl03_fw/main/firmware/mgl03_update.sh && sh /tmp/update.sh
```

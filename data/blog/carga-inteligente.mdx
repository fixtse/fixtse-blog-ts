    Utilizando la aplicación oficial de Home Assistant para Android o iOS, añadiremos un sensor a nuestro home assistant para poder apagar el enchufe inteligente cuando el teléfono haya alcanzado el porcentaje de carga que elijas.

    Funciona con cualquier enchufe inteligente que se integre con home assistant.

      Para esta automatización, necesitaremos un enchufe inteligente que cuente con sensor de potencia y se integre con home assistant.

      La automatización vigilará la potencia utilizada por el enchufe inteligente, cuando llegue al valor que selecciones (la potencia utilizada va disminuyendo con forme la batería llega al 100%), el enchufe inteligente se apagará de forma automática.

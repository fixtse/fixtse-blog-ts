## Lista de comandos en Home Assistant

![V11](/static/images/probar-hass/v11.png)

1. Usuario para inicio de sesión: `root` (no requiere contraseña)
2. Comando para ingresar al sistema: `login`
3. Comando para visualizar IP de Home Assistant: `nmcli`

Buscamos el adaptador por defecto y nos fijamos en la propiedad `inet4` que indica el número de IP (**la ip será diferente en cada sistema**).

**Ejemplo:**

![V12](/static/images/probar-hass/v12.png)

- Con este dato, abrimos una pestaña en el navegador, escribimos la IP que encontramos en el paso anterior y agregamos el puerto 8123 (valor por defecto de home assistant) de la siguiente forma

![V13](/static/images/probar-hass/v13.png)

- Dependiendo de la velocidad de nuestra máquina, es probable que nos aparezca el siguiente mensaje:

![V14](/static/images/probar-hass/v14.png)

- Esto es normal, solo deben esperar con la venta abierta a que el sistema termine de cargar.

![V15](/static/images/probar-hass/v15.png)

- Cuando les aparezca esta página, significa que el sistema se encuentra listo y ya pueden iniciar la configuración inicial tal y como muestro en el video.
